# LPPMI UNUSIA 

## Deskripsi Ringkas
Web untuk Lembaga Pengawasan dan Penjaminan Mutu Internal di Universitas Nahdlatul Ulama Indonesia (UNUSIA). Web ini dibangun dengan dukungan kerangka pengembangan web pada Framework CodeIgniter 4.x. Selengkapnya baca [Teknologi](#teknologi) dan [Fitur](#fitur) üëãüèª
